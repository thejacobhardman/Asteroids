# Asteroids
 A clone of Asteroids built using Pygame.

 Controls:
 Up-Arrow / W: Forward
 Down-Arrow / S: Reverse
 Right-Arrow / D: Rotate Right
 Left-Arrow / A: Rotate Left
 Space: Shoot
